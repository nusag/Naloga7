# GLAVNE FUNKCIONALNOSTI SPLETNE STRANI
1. **Prijava in registracija uporabnikov**: spletna stran omogoča prijavo z uporabo uporabniškega računa
2. **Prikaz dinamičnih podatkov**: podatki se nalagajo iz strežnika in se samodejno posodabljajo.
3. **Interaktivni uporabniški vmesnik**: elemente je mogoče filtrirati, iskati ali urejati v realnem času.
4. **Upravljanje vsebin**: omogočeno je dodajanje, urejanje in brisanje objav ali podatkov.
5. **Mobilna odzivnost**: aplikacija je prilagojena za uporabo na različnih napravah, od telefonov do namizij.
